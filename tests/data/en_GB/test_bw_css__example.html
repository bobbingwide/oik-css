When the <code> tag follows a <p> tag use a 14px font and different colours

[bw_css ]p> code { font-size: 14px; color: white; background: #333; }[/bw_css]

Elsewhere display <code> in blue.

[bw_css ]code { color: blue; }[/bw_css]

Use a parameter to cause the CSS to be shown.

[bw_css .]td code b { color: darkblue; }[/bw_css]

td code b { color : darkblue ; }
